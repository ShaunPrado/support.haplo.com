*Haplo Repository* publishes research outputs on the web, and provides administrative workflows and reporting.

Haplo Repository is open source and supporting documentation is published on Haplo's open source documentation web site at "docs.haplo.org/app/repository":https://docs.haplo.org/app/repository.
