h2. Roles set by department, committee or project

 Navigate to the organisation unit, committee, or other record using the navigation.

[IMAGE 0008-f.png Haplo Research Manager home page]

[IMAGE 0010-f.png Committee list]

[IMAGE 0012-f.png Committee record]

Click the *Edit* button.

[IMAGE 0013-f.png Editing the committee record]

Click the - button to remove roles, click the + button then type the user's name to add a role. Then click Save.

Similarly, use the organisational structure navigation to find and edit roles at organisational units.

[IMAGE 0016-f.png Organisational unit record]

[IMAGE 0018-f.png Editing the organisational unit record]


h2. Roles set globally for the application

Click your name in the top right to show the main menu, then choose System management.

[IMAGE 0009-f.png Main menu]

The options available to you in System management will depend on your permissions.

[IMAGE 0004.png System management]

Click Groups, then the group name corresponding to the role.

[IMAGE 0005.png Group information]

Select the members of the group, which can be other groups and individual users, then click "Save changes"

[IMAGE 0006.png Editing group members]
